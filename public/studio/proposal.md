## 5. Mengapa Ruby Web Studio

> Bagian ini bukan jargon kosong. Saya tulis apa adanya.

### 5.1 Tentang Saya

Saya **Afiq Andico**, biasa dipanggil **Ruby**. Lulusan baru IT, sehari-hari kerja sebagai IT Support di Cube Cafe Jimbaran. Sambil kerja, saya membangun Ruby Web Studio sebagai side project — fokus di **website portfolio untuk freelancer, kreator, dan UMKM Indonesia**.

Saya bukan agency besar dengan 50 karyawan. Saya **sendiri, plus beberapa trusted freelancer** untuk copywriting dan shooting kalau diperlukan. Ini artinya:
- Anda komunikasi langsung dengan saya — bukan account manager.
- Harga lebih masuk akal karena tidak ada overhead kantor besar.
- Saya personally yang coding, QA, dan training.

### 5.2 Pendekatan Kerja

1. **Riset dulu, desain kemudian** — saya pahami bisnis Anda sebelum buka Figma.
2. **Transparan soal harga** — ada kalkulator online yang bisa Anda pakai sendiri sebelum deal.
3. **Handover yang proper** — source code, dokumentasi, kredensial. Anda tidak bergantung pada saya selamanya.
4. **Bahasa Indonesia** — meeting, dokumen, komentar kode. Tidak ada jargon yang disembunyikan.

### 5.3 Tools yang Saya Pakai

- **Editor**: VSCode + Git
- **Design**: Figma untuk mockup, referensi visual
- **Project management**: Notion (shared dengan klien)
- **Komunikasi**: WhatsApp untuk harian, email untuk dokumen
- **AI-assisted**: untuk research & boilerplate (transparan, tidak menggantikan kreativitas)
